Svg Playground

Du Svg pour le web.

Svg Shapes :

  • Rectangle : rect
  • Circle : circle
  • Ellipse : ellipse
  • Line : line
  • Polyline : polyline
  • Polygon : polygon
  • Path : path

Les attributs d'une shape peuvent être définis en css/scss :

			
$cuteblue:rgb(94, 166, 233);

#myCircle{
	cx:150;
	cy:150;
	fill:$cuteblue;
	r:50;
}

On peut appliquer des transitions et des pseudo-selecteurs à des shapes svg en css :

	
#myRect{
	x:600px;
	y:28px;
	fill:$coolgreen;
	width:125px;
	height:100px;
	rx:20px;
	transition:all .5s ease-in;
	&:hover{
		fill:darkgreen;
	}
}
